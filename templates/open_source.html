Ανοιχτός Κώδικας

Το metathesi.gr αποτελεί έργο Ελεύθερου Λογισμικού / Λογισμικού Ανοιχτού Κώδικα. Έχει γραφτεί στο Flask Python micro-framework. Δεν είναι τέλειο προγραμματιστικά αλλά βελτιώνεται συνεχώς!
Πριν "ανέβει" και εκτεθεί στο internet, αναπτύχθηκε και δοκιμάστηκε σε ένα LAMP server που έτρεχε Debian Linux. Ο text editor που χρησιμοποιήθηκε είναι ο Atom. Όλα τα παραπάνω είναι Ελεύθερα Λογισμικά / Λογισμικά Ανοιχτού Κώδικα.

Ο πιο πρόσφατος κώδικας είναι διαθέσιμος στη σελίδα του project στο github.

Η βάση δεδομένων που βρίσκεται πίσω από την εφαρμογή είναι διαθέσιμη εδώ.
